Aquí administras todo el contenido del sitio: jugadores, calendario, resultados, noticias, galería, tienda y más. Edita, guarda y descarga el archivo data/db.json.

Cómo publicar tus cambios: 1) Edita aquí. 2) Pulsa Descargar db.json. 3) Reemplaza el archivo data/db.json en tu repositorio de GitHub (o arrástralo a Netlify). El sitio se actualiza solo. Tus cambios se guardan también en este navegador automáticamente.
